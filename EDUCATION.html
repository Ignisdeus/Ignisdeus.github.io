Michael has been designing and creating games for the last six years in an academic environment. These games include top-down space shooters, FPS, RPG(Tabletop), Board games(tabletop), maze, puzzles, casual/hyper-casual, co-op, narrative-driven, twitch reaction, alt controller, roguelike and arcade. Some of these gamers are created for PC, Android and/or AR.

Michael has experience in many different types of input and target devices. This experience allows for quick iteration and design of new and unique input methods.

Michael has released many games on Itch.io. A lot of these games are experiments in player input and player feedback. Joining game jams allows more feedback on games that have unique and interesting input methods. This games jams range in time frame from 3 hours to 3 months.
